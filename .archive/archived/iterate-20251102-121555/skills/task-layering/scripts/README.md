# Scripts Directory

Create skill-specific helper scripts here.

## Guidelines

- Scripts should be mechanical (deterministic operations)
- Use descriptive names (validate-*, generate-*, analyze-*)
- Make executable: `chmod +x script-name.sh`
- Document usage in comments

## Script Types

- **Validation**: Check structure, format, completeness
- **Generation**: Create files from templates
- **Analysis**: Parse and extract information
- **Transformation**: Convert between formats
