## [1.0.0] - 2025-10-31

### Added
- Initial release of iterate plugin
- Task breakdown, layering, and iteration workflow management
- Parallel multi-agent execution support
- Commands:
  - `/iterate:adjust` - Adjust implementation based on feedback or requirements change
  - `/iterate:enhance` - Enhance existing features with improvements and optimizations
  - `/iterate:refactor` - Refactor code for better maintainability and structure
  - `/iterate:sync` - Sync project state - update specs, tasks, and documentation
  - `/iterate:tasks` - Transform sequential tasks into layered tasks with parallel agent assignments
- Agents:
  - `code-refactorer` - Code refactoring for better maintainability
  - `feature-enhancer` - Feature enhancement and optimization
  - `implementation-adjuster` - Targeted code adjustments based on feedback
  - `task-layering` - Task stratification and parallel execution planning
- Skills:
  - `task-layering` - Task complexity stratification and agent assignment patterns
- Comprehensive documentation and examples
- Infrastructure-first task organization
- Complexity-based agent assignment
- Parallel execution optimization
